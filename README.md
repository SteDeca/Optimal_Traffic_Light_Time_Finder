# Optimal_Traffic_Light_Time_Finder
This is a small python program that casually generates waiting times at a traffic light for three different roads, then finds the optimal time

# Why
During an interview for a job, I was asked to find how long my traffic light cycles have to last in order to get the average of the least amount of time a car has to wait while in line.
This is my best solution (at the moment) for this problem. Does it work? yes. Could it better? of course. But is it enough? absolutely.
